# 🔐 Substitution Cipher Tool
Its a simple encryption system, that ensures privacy. Since all of the popular messaging apps dont have end to end encryption anymore.

## ✨ Features

🔒 **Custom Substitution Cipher** - Advanced Unicode character mapping for alphabet and numerals  
🔄 **Bidirectional Encryption** - Both encrypt and decrypt with ease  
📊 **Interactive CLI** - User-friendly command-line interface  
🗝️ **Complete Cipher Mapping** - View the full alphabet and numeric substitution table  
⚡ **Fast & Efficient** - Dictionary-based lookups for instant encoding/decoding  
🎓 **Educational** - Learn cryptography concepts through hands-on implementation

📚 How It Works
The cipher uses a substitution cipher approach where:

Each letter (a-z) is replaced with a unique Unicode character
Each digit (0-9) is replaced with a unique Unicode character
Spaces and punctuation remain unchanged
Encryption converts plaintext → ciphertext
Decryption reverses the process using a reverse mapping
🎓 Educational Disclaimer
⚠️ This tool is for educational purposes only.

This substitution cipher is not cryptographically secure and should never be used for:
