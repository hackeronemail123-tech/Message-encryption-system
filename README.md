рҹ“ҡ How It Works
The cipher uses a substitution cipher approach where:

Each letter (a-z) is replaced with a unique Unicode character
Each digit (0-9) is replaced with a unique Unicode character
Spaces and punctuation remain unchanged
Encryption converts plaintext вҶ’ ciphertext
Decryption reverses the process using a reverse mapping
